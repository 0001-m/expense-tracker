# Expense Tracker

A React app to track your daily expenses with persistent storage.

## Features
- Add expenses with title and amount
- Delete individual expenses
- Total updates automatically
- Data persists on page refresh using localStorage

## Tech Used
- React
- Tailwind CSS
- localStorage API
- Vite

## What I learned
- Managing multiple state variables
- Derived state with reduce instead of storing total separately
- Deleting items from state with filter
- Syncing state with localStorage using useEffect
- Reading from localStorage on app startup
